StreetFit (REST API)

Java MySQL Spring Boot Hibernate Apache Tomcat embebido Maven

StreetFit es una API REST desarrollada con Spring Boot que proporciona servicios CRUD personalizados para gestionar usuarios de tipo Atletas y Entrenadores. El backend está hecho en Java usando Spring Boot y Hibernate/JPA para interactuar con una base de datos MySQL. Además, el proyecto incluye un cliente Android desarrollado en Android Studio con Java, que consume la API mediante Retrofit para la comunicación entre la app y el servidor.

StreetFit (Cliente)

Java XML Retrofit

Cliente-StreetFit es una aplicación móvil Android desarrollada con Android Studio que funciona como cliente REST para consumir una API backend construida con Spring Boot. Esta app se comunica con la API usando Retrofit, una librería que facilita las peticiones HTTP y la conversión de JSON a objetos Java de forma eficiente. La aplicación permite consultar datos desde el servidor, enviar información mediante formularios y manejar errores de red.
